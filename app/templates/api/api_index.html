Newser API Documentation

Welcome to the Newser API documentation. Below you'll find all available endpoints and their usage.

GET

/api/recommendations

Get personalized news recommendations based on query.

Query Parameters:

  • query (optional) - Search term for news (default: "technology")

Sample Response:

{
    "recommendations": [
        {
            "title": "Article Title",
            "url": "https://article-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
GET

/api/trending

Get current trending news articles.

Sample Response:

{
    "trending": [
        {
            "title": "Trending Article",
            "url": "https://article-url.com",
            "image": "https://image-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
GET

/api/top-story

Get the current top story.

Sample Response:

{
    "story": {
        "title": "Top Story Title",
        "url": "https://article-url.com",
        "image": "https://image-url.com",
        "source": "News Source",
        "publishedAt": "2024-01-20T12:00:00Z",
        "description": "Article description"
    }
}
GET

/api/popular

Get popular news articles.

Sample Response:

{
    "popular": [
        {
            "title": "Popular Article",
            "url": "https://article-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
