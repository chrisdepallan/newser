Newser API Documentation

Welcome to the Newser API documentation. Below you'll find all available endpoints and their usage.

GET

/api/recommendations

Get personalized news recommendations based on query.

Query Parameters:

  • query (optional) - Search term for news (default: "technology")
  • max (optional) - Maximum number of recommendations to return (default: 5)

Sample Response:

{
    "recommendations": [
        {
            "title": "Article Title",
            "url": "https://article-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
GET

/api/trending

Get current trending news articles.

Sample Response:

{
    "trending": [
        {
            "title": "Trending Article",
            "url": "https://article-url.com",
            "image": "https://image-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
GET

/api/top-story

Get the current top story.

Sample Response:

{
    "story": {
        "title": "Top Story Title",
        "url": "https://article-url.com",
        "image": "https://image-url.com",
        "source": "News Source",
        "publishedAt": "2024-01-20T12:00:00Z",
        "description": "Article description"
    }
}
GET

/api/popular

Get popular news articles.

Sample Response:

{
    "popular": [
        {
            "title": "Popular Article",
            "url": "https://article-url.com",
            "source": "News Source",
            "publishedAt": "2024-01-20T12:00:00Z"
        }
    ]
}
GET

/api/article-detail

Get detailed information about a specific article.

Query Parameters:

  • article (required) - Full URL of the article to fetch details for

Sample Response:

{
    "article": {
        "title": "The work sickness timebomb that risks a lost generation of workers",
        "url": "https://www.bbc.com/news/articles/c99vz4kz5vzo",
        "content": "The UK is sick. It's much sicker than other similar countries, and the situation is getting worse, snowballing into a health, social, medical, economic, and potential budgetary crisis...",
        "author": "BBC News",
        "publishedAt": "2024-02-20T12:00:00Z",
        "tags": ["Health", "UK economy", "Labour Party", "Mental health", "Coronavirus"],
        "relatedArticles": [
            {
                "title": "Unemployment rises as pay growth slows again",
                "url": "https://www.bbc.com/news/business-68311416",
                "publishedAt": "2024-02-20T10:00:00Z"
            }
        ]
    }
}
